Transnote インストール手順
==========================

■ まずお読みください

Transnote は未署名で配布しています。macOS のセキュリティ（Gatekeeper）により、
インストール.command をダブルクリックすると次のような警告が出ることがあります。

  「インストール.command」は開いていません
  （ゴミ箱に入れる / 完了 だけのダイアログ）

これはマルウェアではなく、Apple が開発元を検証できないという通知です。
右クリック →「開く」でも通らない場合があります（macOS の新しいバージョンでよくあります）。

その場合は、下記【方法 A】のターミナルからの実行を使ってください。

■ インストール手順

【方法 A】ターミナルから実行（推奨・いちばん確実）

1. アプリケーション → ユーティリティ → ターミナル.app を開く
   （Spotlight で「ターミナル」と検索しても開けます）
2. 次の1行目を貼り付けて Enter を押す:

xattr -cr "/Volumes/Transnote/"

3. 次の1行目を貼り付けて Enter を押す:

bash "/Volumes/Transnote/インストール.command"

※ DMG のボリューム名が Transnote でない場合（例: Transnote 1）は、
  Finder 左サイドバーに表示されている名前に合わせてパスを変更してください。

  例: bash "/Volumes/Transnote 1/インストール.command"

4. インストールが完了すると Transnote が起動します。
5. Finder で DMG を取り出してください（マウントしたままだとアプリが複数見えます）。

【方法 B】右クリック →「開く」

方法 A でうまくいかない場合、またはターミナルを使いたくない場合に試してください。

1. インストール.command を右クリック（Control + クリック）
2. メニューから「開く」を選ぶ
3. 確認ダイアログで「開く」を押す

※ ダブルクリックでは実行できません。
※ 最近の macOS では方法 B でもブロックされることがあります。

【方法 C】システム設定から許可

1. 警告ダイアログは「完了」で閉じる（「ゴミ箱に入れる」は押さない）
2. システム設定 → プライバシーとセキュリティ を開く
3. 下の方に「インストール.command」に関する「このまま開く」があれば押す

■ インストール後

起動しない場合は、Applications フォルダ内の Transnote を
右クリック →「開く」を試してください。

■ 手動インストール（方法 A〜C がすべて失敗した場合）

1. 実行中の Transnote を終了する
2. Applications 内の Transnote.app（および LocalTranscriber.app）を削除する
3. DMG 内の Transnote を Applications フォルダへドラッグする
4. Applications 内の Transnote を右クリック →「開く」

旧バージョンが残っていると Transnote 2.app などの重複ができるため、
通常は方法 A のインストール.command を推奨します。
