Transnote インストール手順
==========================

■ インストール.command が開けない場合

macOS のセキュリティにより「開いていません」と表示されることがあります。
GitHub からダウンロードした未署名ファイルでは正常な動作です。

【方法 A】右クリック →「開く」（推奨）
1. インストール.command を右クリック（Control + クリック）
2. メニューから「開く」を選ぶ
3. 確認ダイアログで「開く」を押す

【方法 B】ターミナルから実行
1. ターミナル.app を開く
2. 次の2行を順に貼り付けて Enter を押す:

xattr -cr "/Volumes/Transnote/"
bash "/Volumes/Transnote/インストール.command"

※ DMG のボリューム名が Transnote でない場合は、Finder サイドバーの表示名に合わせてパスを変更してください。

■ インストール後

インストールが完了すると Transnote が起動します。
起動しない場合は、Applications フォルダ内の Transnote を右クリック →「開く」を試してください。

■ 手動インストール

インストール.command の代わりに、DMG 内の Transnote を Applications フォルダへドラッグしても構いません。
この場合、旧バージョンが残っていると重複する可能性があるため、通常はインストール.command の利用を推奨します。
