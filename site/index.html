macOS 14 以降

Transnote

ローカルの音声ファイルを WhisperKit で文字起こし。処理はすべて端末内で完結し、クラウド送信は行いません。

最新版をダウンロード リリース一覧 GitHub

できること

  • ローカル音声ファイルのドラッグ&ドロップまたはファイル選択
  • WhisperKit によるオンデバイス文字起こし
  • TXT / SRT / VTT / JSON へのエクスポート
  • モデルのダウンロードと選択

インストール

  1. 最新版 DMG をダウンロードします（ファイル名は Transnote.dmg）。
  2. DMG を開き、インストール.command をダブルクリックします（初回は右クリック → 開く が必要な場合があります）。
  3. インストールスクリプトが Applications 内の旧バージョンを削除してから新バージョンを配置します。
  4. 未署名ビルドのため、初回起動時に Gatekeeper の警告が出る場合は、Finder で Transnote を右クリック → 開く を選んでください。
  5. 初回利用時に Whisper モデルのダウンロードが必要です。画面の案内に従ってください。

詳細はリポジトリ内の インストール手順 を参照してください。

プライバシー

Transnote は音声データをクラウドへ送信しません。文字起こし処理は端末上の WhisperKit で実行されます。 モデルの取得時のみ、Hugging Face などの配布元へネットワーク接続が必要です。
