Transnote

ローカルの音声ファイルを WhisperKit で文字起こし。処理はすべて端末内で完結し、クラウド送信は行いません。

最新版をダウンロード GitHub

できること

  • ローカル音声ファイルのドラッグ&ドロップまたはファイル選択
  • WhisperKit によるオンデバイス文字起こし
  • TXT / SRT / VTT / JSON へのエクスポート
  • モデルのダウンロードと選択

インストール

未署名配布のため、インストール.command をダブルクリックすると「開いていません」と表示されることがあります。右クリック → 開く でも通らない場合は、ターミナルからの実行がいちばん確実です。

  1. 最新版 DMG をダウンロードします（ファイル名は Transnote.dmg）。
  2. DMG を開き、初めにお読みください.txt を読みます。
  3. ターミナルを開き、次の2行を1行ずつ実行します（ボリューム名は DMG により Transnote など）: 
    xattr -cr "/Volumes/Transnote/"
bash "/Volumes/Transnote/インストール.command"
  4. インストールスクリプトが Applications 内の旧バージョンを削除してから新バージョンを配置します。
  5. DMG を Finder から取り出します（マウントしたままだとアプリが複数表示されます）。
  6. 初回起動時に Gatekeeper の警告が出る場合は、Applications 内の Transnote を右クリック → 開く を選んでください。
  7. 初回利用時に Whisper モデルのダウンロードが必要です。画面の案内に従ってください。

詳細は リリース一覧 またはリポジトリ内の インストール手順 を参照してください。

プライバシー

Transnote は音声データをクラウドへ送信しません。文字起こし処理は端末上の WhisperKit で実行されます。 モデルの取得時のみ、Hugging Face などの配布元へネットワーク接続が必要です。
